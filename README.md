# TASKR

TASKR is a ward task coordination app for inpatient teams. It helps organize practical jobs, prescribing follow-ups, referrals, and discharge tasks in one place.

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
npm install
npm run dev
```

App runs on `http://localhost:8080/` by default.

## Build

```sh
npm run build
```

## Lint

```sh
npm run lint
```
